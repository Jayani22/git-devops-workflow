# Git DevOps Workflow

## Project Overview

This project demonstrates Git best practices commonly used in DevOps teams.

It includes:

- Branching strategy
- Feature development workflow
- Pull Requests
- Git tags
- Markdown documentation
- GitHub collaboration

---

## Branching Model

- **main** – Production-ready code
- **dev** – Integration branch
- **feature/*** – Feature development
